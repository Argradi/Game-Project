# Average Adventure ⚔️

**Average Adventure** es un juego de acción y supervivencia en 2D donde el objetivo es simple: sobrevivir el mayor tiempo posible mientras acumulas puntos recogiendo monedas y eliminando enemigos.

---

## 🎮 Mecánicas del Juego

El juego se desarrolla en un escenario compuesto por el **suelo y tres plataformas elevadas** que el jugador debe utilizar estratégicamente para moverse y evitar el contacto con los enemigos.

### Sistema de Puntuación
La puntuación se calcula de forma dinámica:
* **Supervivencia:** +1 punto por cada segundo transcurrido.
* **Recolección:** +5 puntos por cada moneda recogida.
* **Combate:** +10 puntos por cada enemigo eliminado con el disparo.

### Flujo de Navegación
1.  **Pantalla de Inicio:** Presenta el título "Average Adventure".
    * **Botón Jugar:** Inicia la acción inmediatamente.
    * **Botón Controles:** Abre una sección informativa con los comandos de movimiento y disparo.
2.  **Game Over:** Se activa al chocar con un enemigo. Muestra la puntuación total alcanzada y ofrece dos opciones:
    * **Volver a intentar:** Reinicia la partida desde cero.
    * **Inicio:** Regresa al menú principal.
